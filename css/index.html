CSS拆分示例

原始CSS文件已被拆分为多个模块化的CSS文件

拆分的CSS文件

  • base.css - 基础样式，包含CSS变量、通用样式和基础设置
  • components.css - 组件样式，包含各种UI组件的样式
  • layout.css - 布局样式，包含布局相关的样式
  • animations.css - 动画样式，包含所有动画和过渡效果
  • responsive.css - 响应式样式，包含所有媒体查询
  • vendors.css - 第三方库样式，包含Waves等第三方库的样式
  • utilities.css - 工具类样式，包含各种实用工具类
  • themes.css - 主题样式，包含不同主题的样式
